# Soma Studio - Project Guidelines

## Overview

Building a high-performing, beautifully designed website for **Soma Studio**, a yoga studio in **Plettenberg Bay, South Africa**, owned and operated by **Ceire Cunningham**.

The site should reflect the serene, grounded nature of yoga while feeling modern and professional. Target audience: locals and visitors in Plettenberg Bay looking for yoga classes and fitness.
